My name is Tobias Bernard. I design brands, interfaces and interactions.

I live in Merano, Italy and study Computer Science at UniBZ. You can read my Blog, look at my Portfolio, fork my code on Github, follow me on Diaspora or Contact me directly.

I almost exclusively use free software on free systems, not only because I think it is the only ethical way to use software, but also because this gets me the best possible computing experience available anywhere.

My operating system of choice is Ubuntu, because it is the most worry-free GNU/Linux distribution. On top of it I use the GNOME Shell desktop because it is by far the best desktop experience of any platform (for example ,the way it handles multitasking and notifications, as well as its extension system are unmatched).

I don't use many different applications, but those that I do use, i use a lot. Since I almost all of my design work is vector, I use Inkscape the most, but I also use GIMP for image manipulations and Scribus for print layouts (although I try to use Inkscape as much as possible, because its UI is better). Other applications I use include Firefox for browsing the web, Scribes for coding/writing and Empathy for instant messaging.
